# Drivers API

## Description

API that shows all the drivers salary

## How to Run

1. Clone the repository

    ```bash
    git clone https://github.com/JordanMarcelino/drivers-backend.git
    ```

2. Move into the directory

    ```bash
    cd drivers-backend
    ```

3. Dockerize the application

    ```bash
    docker compose up -d --build
    ```

## How to Stop

```bash
docker compose down
```
